Le jeu de Kimmi

L’objectif est d’aider le personnage, contrôlable avec les flèches du clavier, à rejoindre le trésor situé en haut à droite de l’écran en évitant les obstacles. Pour contrôler le personnage, il faut d’abord cliquer dans le cadre du jeu afin de l’activer.

Mot de l'élève pour présenter son jeu : {À venir}
